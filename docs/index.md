# Getting started

Guided is a CLI tool designed to amplify the work of engineers using AI.

## Why did we write this?

Why did we write this?  Simple.  AI is being leveraged increasingly to build new software but as veteran engineers we understand that maintaining software is increasingly difficult throughout the lifecycle of a project.  

We wanted to build an workflow that could help engineers build better software and provide the tools with which to do it.  

## Usage

If you haven't done so already install and configure the CLI tool by following the instructions in the [setup](setup/index.md) guide.

Start a TUI and chat directly

```bash
guide chat
```

Or use a command line interface

```bash
echo "What files are in this directory?" | guide chat
```
