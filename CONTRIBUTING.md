CONTRIBUTING.MD
---------------

Here's how to contribute.  

Fork the repository.  

Then clone your fork and create a branch for your feature or bug fix.

```
    git clone git@github.com:your-username/guided.git
    cd guided
    git remote add upstream git@github.com:guided-code/guided.git
    git checkout -b feature-or-bugfix-branch
```

Make your changes, commit, push your branch.  

Then create an Pull Request in Github referencing the upstream branch and issue if any.
